# 🐦 Boids Simulation
 
A **Boids flocking simulation** built in **Unity (C#)** as an introduction to the engine.
 
Boids is an artificial life algorithm developed by Craig Reynolds in 1986 that simulates the flocking behavior of birds. Each agent follows three simple rules:
 
- **Separation** — avoid crowding neighbors
- **Alignment** — steer towards the average direction of neighbors
- **Cohesion** — steer towards the average position of neighbors
 
These three rules combined produce emergent, lifelike flocking behavior.
